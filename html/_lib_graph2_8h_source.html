460// qui utilisent cette DLL. De cette manière, les autres projets dont les fichiers sources comprennent ce fichier considèrent les fonctions
461// BEAUH_API comme étant importées à partir d'une DLL, tandis que cette DLL considère les symboles
462// définis avec cette macro comme étant exportés
