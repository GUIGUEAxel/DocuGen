\doxysubsection{Detailed Description}
Classe représentant un labyrinthe. 

Cette classe permet de stocker un labyrinthe sous forme de cases et propose des méthodes pour définir le point de départ, le point d’arrivée et avancer dans la résolution du labyrinthe.
