\doxysubsection{Detailed Description}
Représente une case du labyrinthe. 

Une case peut être connectée à ses voisins (haut, droite, bas, gauche) et stocke son état (visitée ou mauvais choix). Elle permet également d\textquotesingle{}être affichée graphiquement.
