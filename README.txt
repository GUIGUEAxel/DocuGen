Une fois le dépôt cloné, ouvrez le dossier avec Visual Studio ou un autre IDE permettant d'exécuter les différents fichiers. Ensuite, cliquez sur "TP 13bis.sln" pour l'ouvrir. Un message apparaîtra, appuyez sur "Continuer", puis cliquez sur "prog.cpp". Enfin, appuyez sur "Ctrl + F5" pour exécuter le programme. À ce stade, la fenêtre devrait s'ouvrir, et vous pouvez utiliser la touche "Espace" pour déplacer le personnage.
